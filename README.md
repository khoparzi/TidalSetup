# TidalSetup

Currently SampleLoader is only compatible with tidal-1.9.5. It can be disabled by commenting out line 14 in init.hs if a newer version (incompatible) of tidal is being used.

I currently install Tidal with 1.9.5 with ```cabal v1-install tidal-1.9.5 --allow-newer=containers``` to go past dependency issues
